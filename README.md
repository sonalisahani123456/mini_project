**Mini Project: Sidebar Menu UI**

This is a simple mini project created for practice using only HTML and CSS.

**🚀 Description**

This project is a responsive sidebar navigation menu with a modern UI.
It includes a toggle button to open and close the sidebar, along with menu items and social media icons.

**🛠️ Technologies Used**
HTML
CSS

**✨ Features**
Toggle sidebar using checkbox trick
Smooth transitions and animations
Icons using Font Awesome
Clean and minimal UI design

**📂 Project Structure**
index.html → Main structure of the webpage
style.css → Styling and layout
photo.jpg → Background image

**🎯 Purpose**
This project is built for practice and learning CSS concepts like:

Positioning
Flexbox
Transitions
UI design basics
